                                                  Kelly Aaron 3 years ago

                                                  Hi Aaron,

                                                  That is a good point. Instead of giving the p selector padding, it would be better to create a class that just takes up enough space to account for the navigation bar, without making the paragraph tag unusable. I have revised the code, and you can see that there is now a class called "page-padding" that looks like this:

                                                  .page-padding {height:90px; width:100%;}

                                                  Now inserting the [html]<div class="page-padding"></div>[/html] directly after the name section (such as [html]<a name="portfolio" rel="nofollow"></a>[/html]), will push the page content down to where it needs to be.

                                                  Thanks for pointing that out!

                                                                                insitedesignlab Author Yorick 2 years ago

                                                                                Hey Yorick!

                                                                                If you're looking to separate the Javascript into an external file that's totally do-able.

                                                                                Before we get started make sure you aren't just copy / pasting the entire thing into a JavaScript file.

                                                                                That first line is calling jQuery and needs to go in your .HTML or .PHP file at the bottom before the </body>

                                                                                Second, make sure you remove the <script> tags from around the rest of the code. Then paste that into an external JavaScript file.

                                                                                As a general rule you want to always:

                                                                                1. Put CSS In the HEAD
                                                                                2. Put JavaScript in the footer. Or right next to the &lt;/body&gt; tag

                                                                                So your code should look like this:

                                                                                     &lt;script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.5.2/jquery.min.js"&gt;&lt;/script&gt;
     &lt;script type="text/javascript" src="autoscroll.js"&gt;&lt;/script&gt;
     &lt;/body&gt;

                                                                                Does that help?
